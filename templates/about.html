About the project:

The project was created by David, Rahaf, Fabian, and Akram. The prototype automates the process of sorting and storing items by color, using various technological components including a robot arm, PLC, Raspberry Pi with a camera, conveyor belt, cloud services, a database, and a web platform. The camera identifies object colors and positions, and the conveyor moves objects to the robot arm for sorting and storage. Users can place orders for separate sets of boxes via a web application, and there's also a purpose-built computer application at the robot's station for manual control, allowing users to start and stop the process.
